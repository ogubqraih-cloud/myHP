# CLAUDE.md — myHP プロジェクト

## プロジェクト概要

チャットボット付きのパーソナルホームページ。自然をテーマにしたモダンなデザインで、AIエンジニアのポートフォリオサイトとして機能する。現在は `index.html` 単一ファイル構成。

## 基本ルール

### 言語
- **返答はすべて日本語で行う**
- コードのコメントも日本語で書く（英語コメント禁止）

### デザイン方針
- **モダンで洗練されたデザイン**を維持する
- 現行のデザイントークン（CSS変数）を踏襲する：
  - カラーパレット：`--ocean-*`（青系）、`--forest-*`（緑系）、`--sand`（背景）
  - グラスモーフィズム（`backdrop-filter: blur`、半透明背景）を活用
  - 角丸：カード類は `border-radius: 20px〜24px`
  - アニメーション：控えめなホバーエフェクト（`translateY(-3px〜-4px)`）
- 新しいUIコンポーネントを追加する際も、既存のデザイントークンを使用する

### レスポンシブ対応
- **スマートフォンでの表示を必ず確認・対応する**
- ブレークポイントの目安：`max-width: 600px`（スマホ）、`max-width: 900px`（タブレット）
- `clamp()` や `min()` を使った流体的なサイジングを優先する
- タッチ操作を考慮したUI（ボタンのタップ領域は最低 44px）

### チャットボット
- チャットボットはページ右下に**フローティングボタン**として実装する
- チャットウィンドウは画面を占有しすぎないよう `max-width: 380px`、`max-height: 520px` 程度に収める
- スマホでは全画面または大部分を占める形にレイアウトを切り替える
- Claude API（`claude-sonnet-4-6` モデル）を使用することを想定している

## ファイル構成

```
myHP/
├── index.html      # メインページ（現在すべてのコードを含む）
├── CLAUDE.md       # このファイル
└── （今後追加予定）
    ├── style.css   # CSSを分離する場合
    ├── chat.js     # チャットボットロジック
    └── assets/     # 画像・アイコン類
```

## コーディング規則

- **コメントは日本語のみ**。WHYが自明でない箇所にだけ付ける
- インラインスタイルは避け、CSSクラスで管理する
- JavaScriptは `const`/`let` を使用、`var` は禁止
- APIキーや秘密情報は絶対にHTMLやJSに直書きしない（環境変数またはバックエンド経由）

## やってはいけないこと

- 既存のカラーパレット・フォント（Zen Maru Gothic / Nunito）を無断で変更しない
- デスクトップ表示のみ確認してスマホ対応を怠らない
- 英語でコメントを書かない
- 派手すぎるアニメーションや過剰な装飾を追加しない（シンプルさを保つ）
